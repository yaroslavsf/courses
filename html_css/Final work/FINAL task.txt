Контрольное задание: "Страница мои итоги курса по html\css"
Цель: Студенту необходимо создать страницу о себе, на которой отображаются навыки полученные в ходе прохождения курса html\css.
Страница состоит из заголовка, основной части и подвала. 
1) Заголовок занимает 1/4 экрана, разбит на 2 блока: блок с фотографией , блок с  ФИО и краткой информацией. 
Требования к заголовку: 
1.1)Фотография размещена по центру своего блока и занимает всю его область с небольшими отступами по краям. Имеет рамку.
Само изображение значения не имеет(можно взять из интернета). 
1.2)ФИО располагается в верхней части своего блока по центру. Занимает одну строку.( Можно использовать инициалы)
1.3)Текст имеет минимум 2 абазаца, вырвнивание по ширине. Между абзацами отступ высотой в одну строку текста.
В тексте разным цветом должны быть выделены "ключевые" слова, минимум 3. (Можно использовать стандартный "текст рыбу".)
2)Основная часть страницы:
Имеет внешний вид как у MyStat.
Состоит из 4 "карточек", (см.скриншот) Внешний вид, цветовое оформление тоже берется из MyStat. 
2.1) Карточка "введение в html" выглядит так-же как карточка "успеваемость". Отображает насколько хорошо студент знает технологию (например 4 столбика: Адаптив, Flex-box, Медия запросы, bootstrap.)
И так-же как на карточке "успеваемость" содержит блок средний балл (но в myStat средний балл состоит из 2-х блоков, за месяц и за все время. В задании же нужен 1 блок. Сами числа значения не имеют)
2.2) Карточка анимация. Содержит пример анимации. По центру карточки вращающийся блок (картинка, просто цветной блок и т.д. ).Анимация должна быть зацикленой.
	Бонус: если при наведении мышкой, анимация начинает вращатся быстрее\медленнее.
2.3) Карточка таблицы. Содержит таблицу. Похожа на карточку "календарь". Требование: обязательно использовать хотябы 1 colspan, rowspan.
2.4) Карточка Flex похожа на блок "Ваши награды" из MyStat. Содержит такое-же наполнение. см. скриншот 2. Важно: картинки и текст произвольный. Бонус: использовать вместо картинок fontawessome.
3) Подвал. Маленький черный блок внизу страницы. Содержит имя-фамилию и значет копирайта. 

Пункты выполнять по очереди. Необходимо за занятие выполнить как-можно больше пунктов.Если возникают затруднения с одним из пунктов, (не знаете как сделать слишком долго сидите над ним), то переходите к след. пункту.
